# **App Name**: Connectify

## Core Features:

- Instant Matching System: Real-time discovery engine that pairs users based on text or video preferences using Socket.IO.
- P2P Video Streaming: Secure, high-quality video and audio calls using WebRTC with dedicated camera/mic toggle controls.
- AI Text Moderation: A safety tool that automatically filters spam and harmful language in real-time messages to ensure a positive community.
- Firebase User Authentication: Supports secure sign-ins via Google, Email, and Guest accounts stored in Firestore.
- Live Messaging & Presence: Dynamic text chat with typing indicators and online user counters for a connected experience.
- Moderation & Safety Suite: Allow users to instantly report or block strangers, with server-side enforcement protocols.
- Responsive Video Interface: Optimized layouts for desktop and mobile devices featuring stranger labels and fluid state transitions.

## Style Guidelines:

- Primary Color: Deep Electric Indigo (#4F46E5) to represent connection and technological vibrancy.
- Background: Ultra-clean Cool Gray (#F3F4F9), maintaining a 15% saturation of the primary hue for a cohesive light-mode aesthetic.
- Accent: Bright Azure (#3B82F6) used for actionable UI elements like the 'Start Chat' buttons to differentiate interaction points.
- A pairing of 'Space Grotesk' for bold, tech-forward headlines and 'Inter' for readable, objective body text.
- Minimalist line icons with 2px strokes, emphasizing clarity and mobile-friendly touch targets.
- A split-screen focus for video chat with an overlay-style messaging drawer to maximize visual real estate.
- Soft spring transitions for component entries and pulse effects on the 'Live' indicators to denote real-time connectivity.
